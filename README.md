							<div class="subSectionDesc">這個遊戲一共有五種地形。<br><ul class="bb_ul"><li><b>海岸：</b><br>- 第一種地區，也是遊戲的起始地區。<br>- 一個平靜的海岸線，有容易攀爬的石頭和低危物品像是無毒的水母。<br>- 最適合學習基礎的遊戲玩法。<br>- 這個區域沒有時限，可以慢慢來探索周遭的行李和食物。<br>- 你可以找到一本指南書，裡面有寫如何控制角色以及基礎的遊戲玩法。<br><br></li><li><b>熱帶</b><br>- 遊戲的第二種地區。<br>- 一片密集且潮濕的叢林。<br>- 出現新的遊戲玩法，像是盪藤蔓、植物陷阱和下雨。<br>- 這個地方時不時會下雨，導致更難攀爬，所以下雨時最好等雨停再行動。<br>- 你也可以用藤蔓來輔助攀爬，但要注意使用藤蔓也會減少體力。<br><br></li><li><b>雪山</b><br>- 遊戲的第三種地區。<br>- 一座危險的雪山。<br>- 冰會減少你的握力並累積冰凍傷害。<br>- 強風會把玩家吹得到處跑，所以你需要找溫暖的地方待著。風會從隨機的方向吹來，所以要在結冰之前移動到安全的地方。<br>- 在這個地區可以找到鎖鏈（類似熱帶區域的藤蔓）和一些已經釘在牆上的岩釘。<br><br></li><li><b>火山</b><br>- 遊戲的第四種地區。<br>- 一個充滿岩漿的火山區域。<br>- 小小噴發就可能引起岩漿潮。如果你站在噴發的區域就會燒傷。<br>- 地上有橘色點標示的話代表很快要爆發了，要趕快躲避。<br>- 這個區域也有鎖鍊。<br><br></li><li><b>火山窯</b><br>- 遊戲的第五種地區。<br>- 火山內部的最後一關，一條岩漿包圍的垂直上坡。<br>- 這是遊戲裡最難的地區。你需要攀爬前謹慎規劃路線，否則會迷路。<br>- 這裡很容易燒傷（跟火山地區一樣），所以最好帶一些治療或防護物品。<br><br></li><li><b>山頂</b><br>最後的地形。最終的目標—山頂。</li></ul><br><div style="clear: both"></div>
